# Fillit project - arsciand & saneveu @ 42

Fillit is a tetrimonos map resolver. 

Usage : 

```make```

And

```./fillit [map.txt]```

Exemple :

Input map ->

```C
...#
...#
...#
...#

....
....
....
####

.###
...#
....
....

....
..##
.##.
....

....
.##.
.##.
....

....
....
##..
.##.

##..
.#..
.#..
....

....
###.
.#..
....
```

Output ->
```C
./fillit map
ABBBB.
ACCCEE
AFFCEE
A.FFGG
HHHDDG
.HDD.G
```
